# Risks — ARES-STARSHIP V3.1

## Critical Risks
1. **Isp < 880s** → requires Mars refuel
2. **Core temp < 2900K** → Isp drops to 850s
3. **Dry mass > 210t** → margin lost
4. **CNEN denial** → no nuclear launch from Brazil

## Mitigations
- 4-engine redundancy
- 833 m/s margin
- NTR only above 400km
- Alcântara equatorial advantage
